## 🚨 快速问题定位

### 编译问题
```bash
# 症状：编译失败
# 解决：清理重建
idf.py clean
idf.py build
```

### 烧录问题
```bash
# 症状：烧录失败
# 解决：检查端口和驱动
idf.py -p COM10 flash
```

### 运行问题
```bash
# 症状：程序无输出
# 解决：检查串口监控
idf.py -p COM10 monitor
```

## 🛠️ 调试工具

### 软件调试工具
- **ESP-IDF Monitor**: 串口日志监控
- **GDB调试器**: 程序断点调试
- **Heap Trace**: 内存泄漏检测
- **Task Monitor**: 任务状态监控

### 硬件调试工具
- **万用表**: 电压、电流、电阻测量
- **示波器**: 信号波形分析
- **逻辑分析仪**: 数字信号时序分析
- **CAN分析仪**: CAN总线协议分析
