## 🔧 硬件架构概览

### 核心组件
- **ESP32-WROOM-32**: 主控制器
- **CAN收发器**: 电机通信接口
- **SBUS接收器**: 遥控信号输入
- **电源管理**: 5V/3.3V转换
- **状态指示**: RGB LED指示灯

### 接口分配
- **UART0**: 调试串口
- **UART1**: CMD_VEL接收
- **UART2**: SBUS信号接收
- **TWAI**: CAN总线通信
- **GPIO**: 状态LED和控制信号

| 参数 | 规格 | 说明 |
